🚀 How to use this site without getting lost

Use the top navigation for the study stage, and use the control panel to filter by lecture or switch study mode. Recommended order: High Yield → Flashcards → Wrong Questions → Exam Simulator → Cheat Reviews.

1️⃣ High Yield

Start with MCQs and instructor-emphasized topics. These are the most likely exam ideas.

2️⃣ Filter Lecture

Pick L1–L6 from the dropdown when you want focused studying without noise.

3️⃣ Practice

Use flashcards and fill blanks to force active recall, not passive reading.

4️⃣ Weak Mode

Review what you got wrong. This is the fastest way to raise your score.

5️⃣ Exam Mode

Run the simulator after reviewing. Use Prediction mode for last-minute studying.

🔥 Prediction = likely exam concepts 🎯 Weak = wrong questions only ⚡ Auto Flow = continue unfinished lecture 🌐 All = full course view
